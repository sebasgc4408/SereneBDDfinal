# Tech Stack: Serene

## Frontend
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** 
  - Tailwind CSS (Utility-first layout)
  - Vanilla CSS Modules (Core "Premium Clinical" components for maximum polish)
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Backend & Database
- **Platform:** Convex (Real-time backend, document store, and serverless functions)

## Authentication
- **Provider:** Clerk (Passwordless email-link authentication)

## Infrastructure & Integrations
- **Calendar:** Google Calendar API (via Clerk OAuth / Google Cloud Console)
- **Deployment:** Vercel (Recommended for Next.js)

## Development Tools
- **Testing:** Vitest / React Testing Library (Recommended for Next.js)
- **Linting:** ESLint
- **Formatting:** Prettier
